## Переподписать N последних commit можно с помощью интерактивного перемещения базы. Это создаст новые копии коммитов с обновленной цифровой подписью.

1. Запустите перебор последних трех коммитов:
```bash
git rebase -i HEAD~3
```
2. В открывшемся текстовом редакторе замените слово pick на edit (или e) для всех строк с нужными вам тремя коммитами. Сохраните и закройте файл.

3. Git остановится на первом из выбранных коммитов. Обновите его подпись командой:
```bash
git commit --amend --no-edit -S
```
4. Перейдите к следующему коммиту:
```bash
git rebase --continue
```

# Повторяйте шаги 3 и 4, пока все три коммита не будут переподписаны.
